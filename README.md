# Analytics Table Dashboard

Интерактивная таблица с динамическим графиком на чистом JavaScript.

## О проекте

Проект представляет собой аналитическую таблицу с показателями:

- Текущий день
- Вчера (значение + % изменения)
- Этот день недели

При клике на строку раскрывается график с динамикой за неделю.

График реализован с помощью Chart.js.
Таблица создаётся полностью через JavaScript (без HTML-разметки вручную).

---

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js

---

## Функциональность

- Динамическое создание таблицы
- Подсветка столбца "Текущий день"
- Отображение процента изменения к предыдущему дню
- Цветовая индикация роста / падения
- Раскрывающийся график по клику
- Минималистичный график (без сетки и подписей)

---

## Что реализовано

- Работа с DOM без фреймворков
- Расчёт процентов изменения
- Условная стилизация ячеек
- Управление состоянием открытого графика
- Настройка Chart.js

---

## Запуск

Открыть `index.html` в браузере.

Никакой сборки и зависимостей не требуется.
